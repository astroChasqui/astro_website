Qoyllur-quipu (q2) is a Python package that helps astronomers to determine atmospheric parameters of stars using the iron line excitation/ionization equilibrium technique. It can also be used to measure elemental abundances and other fundamental stellar parameters such as mass, radius, luminosity, and age. It employs the spectrum synthesis code MOOG, a grid of model stellar atmospheres (Kurucz or MARCS), and a theoretical isochrone set (Yonsei-Yale). A web-based extension of this software (q2web) is under development. The source code is available on GitHub.

In short, q2 takes as input two comma-separated values files: one with star data such as initial guess values for Teff, log g, etc., and another with spectral line data, i.e., atomic parameters and equivalent widths of any number of stars. A simple series of commands will then solve for the stellar parameters that satisfy excitation/ionization equilibrium (in this example using MARCS models and the data for 'Sun' as reference, i.e., a line-by-line differential analysis):

>>> import q2
>>> data = q2.Data('stars.csv', 'lines.csv')
>>> sp = q2.specpars.SolvePars(grid='marcs')
>>> q2.specpars.solve_all(data, sp, 'solution.csv', 'Sun')

The q2 code is much more flexible than this simple example suggests (and in principle easily extendable). This Tutorial has more information on the many different options that are available.

Qoyllur and quipu are words in Quechua, a language currently spoken by millions of people in the highlands of South America. It was the official language of the Inca empire and one of the present-day official languages of Peru. The word qoyllur means star, while quipu was a device used in accounting, a form of calculator. Thus, qoyllur-quipu roughly translates to ''star calculator.''
